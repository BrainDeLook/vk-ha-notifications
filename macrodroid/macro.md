# Настройка MacroDroid

## Установка

[MacroDroid в Google Play](https://play.google.com/store/apps/details?id=com.arlosoft.macrodroid)

## Создание макроса

### 1. Открой MacroDroid → Добавить макрос

### 2. Триггер — уведомление от ВКонтакте

- Нажми **+** в разделе Триггеры
- Выбери **Уведомление получено**
- Приложение: **ВКонтакте**
- Поставь галочку **Уведомление содержит текст** если хочешь фильтровать

### 3. Действие — отправить webhook в Home Assistant

- Нажми **+** в разделе Действия
- Выбери **HTTP-запрос**
- Заполни:

| Поле | Значение |
|---|---|
| URL | `http://192.168.1.145:8123/api/webhook/vk_new_message` |
| Метод | `POST` |
| Тип содержимого | `application/json` |
| Тело | см. ниже |

**Тело запроса:**
```json
{
  "title": "{notification_title}",
  "text": "{notification_text}"
}
```

`{notification_title}` и `{notification_text}` — это переменные MacroDroid,
они автоматически подставят заголовок и текст перехваченного уведомления.

### 4. Сохрани макрос

Дай название, например **VK → HA**, и сохрани.

### 5. Выдай разрешение на чтение уведомлений

MacroDroid попросит разрешение при первом запуске макроса.
Или вручную: **Настройки Android → Уведомления → Доступ к уведомлениям → MacroDroid**.

## Тест

1. Напиши себе в ВК с другого аккаунта
2. На Android придёт уведомление
3. MacroDroid перехватит и отправит webhook
4. На iPhone придёт push мгновенно
5. Нажатие на push откроет ВКонтакте

## Изменение адреса HA

Если IP домашнего роутера изменился — поменяй URL в макросе MacroDroid.
Текущий адрес: `http://192.168.1.145:8123`

Чтобы IP не менялся — назначь статический IP для HA в настройках роутера
(обычно: DHCP → резервирование по MAC-адресу).
