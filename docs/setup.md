# Инструкция по установке

## 1. Получи токен VK

См. [vk-token.md](vk-token.md)

## 2. Добавь токен в secrets.yaml

В файле `/config/secrets.yaml` на твоём Home Assistant:

```yaml
vk_access_token: "твой_токен_здесь"
```

## 3. Подключи файлы конфигурации

В `configuration.yaml` добавь или убедись что есть:

```yaml
sensor: !include sensors.yaml
automation: !include automations.yaml
```

Либо скопируй содержимое файлов в уже существующие секции.

## 4. Замени имя устройства

В `automations.yaml` найди строку:

```yaml
service: notify.mobile_app_iphone
```

Замени `iphone` на реальное имя твоего устройства. Его можно найти в:
**Настройки → Мобильное приложение → (твоё устройство)**

Обычно выглядит как `notify.mobile_app_iphone_имя` или `notify.mobile_app_имя_телефона`.

## 5. Перезапусти Home Assistant

**Настройки → Система → Перезапуск**

## 6. Проверь сенсор

После перезапуска в **Панели разработчика → Состояния** найди:
- `sensor.vk_unread_count` — должен показывать число
- `sensor.vk_last_sender` — имя последнего отправителя

## 7. Тест уведомления

Напиши себе сообщение в ВК с другого аккаунта — через 30 секунд придёт push.
При нажатии на уведомление откроется мессенджер ВКонтакте.
