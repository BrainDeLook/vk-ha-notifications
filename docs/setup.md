# Инструкция по установке

## 1. Получи токен VK

См. [vk-token.md](vk-token.md)

## 2. Добавь репозиторий в Home Assistant

**Настройки → Дополнения → Магазин дополнений → ⋮ → Репозитории**

Добавь URL:
```
https://github.com/BrainDeLook/vk-ha-notifications
```

## 3. Установи аддон

В магазине дополнений найди **VK Notifications** и нажми **Установить**.

## 4. Настрой аддон

На вкладке **Настройка** заполни:

| Параметр | Значение |
|---|---|
| `vk_access_token` | Токен VK из [vk-token.md](vk-token.md) |
| `ha_url` | `http://homeassistant:8123` |
| `ha_webhook_id` | `vk_new_message` |
| `log_level` | `info` |

## 5. Создай автоматизацию с webhook

Скопируй содержимое `ha-config/automations.yaml` в свои автоматизации.

Замени в строке `notify.mobile_app_iphone` — `iphone` на имя своего устройства.
Найти его можно в: **Настройки → Мобильное приложение → твоё устройство**.

## 6. Запусти аддон

На вкладке **Информация** нажми **Запустить**.
Включи **Запускать при старте** и **Отслеживать**.

## 7. Проверь логи

На вкладке **Журнал** должно появиться:
```
VK Long Poll запущен, ожидаю сообщения...
```

## 8. Тест

Напиши себе сообщение с другого аккаунта — уведомление придёт мгновенно.
При нажатии откроется мессенджер ВКонтакте.
